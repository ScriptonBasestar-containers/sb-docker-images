## What is Groupware?

Groupware includes collaborative software for:
- **Calendars**: Event scheduling and sharing
- **Contacts**: Address book management
- **Tasks**: To-do list management
- **Email**: Collaborative email (webmail)

## CalDAV/CardDAV Standards

AgenDAV uses open standards:
- **CalDAV**: Calendar protocol (RFC 4791)
- **CardDAV**: Contact protocol (RFC 6352)
- **iCalendar**: Event format (RFC 5545)
- **vCard**: Contact format (RFC 6350)
